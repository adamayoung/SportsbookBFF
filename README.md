# Sportsbook BFF - Backend-for-Frontend

![Mainline](https://github.com/adamayoung/SportsbookBFF/workflows/CI/badge.svg)

## Requirements

* Swift 5.5+

## Setup

Create a `.env` file in the project root directory with the following variables:

```bash
LOG_LEVEL=info
TLA_KEY=xxx
# Optional
AUTH_KEY=xxx
```
